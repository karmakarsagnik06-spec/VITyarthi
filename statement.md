
# Attendance Management System - Project Statement

## Problem Statement
Managing student attendance manually often leads to errors, difficulty in maintaining records, and time consumption.  
A simple digital tool is needed to record, view, and search attendance easily using a lightweight and beginner-friendly approach.

## Scope of the Project
- Basic attendance recording (name + date + status)
- Store records locally in a CSV file
- View all attendance entries
- Search for specific students
- Suitable for small classes, labs, or personal use
- Designed for beginners practicing Python file handling

## Target Users
- Students learning Python
- Teachers or tutors managing small groups
- Beginners practicing file handling and functions
- Anyone needing a simple offline attendance tracker

## High-Level Features
- Mark attendance (Present/Absent)
- Display all stored attendance records
- Search attendance by student name
- CSV-based data storage
- Simple menu-driven interface
